# RUL Prediction on NASA C-MAPSS

Remaining Useful Life prediction using deep learning models with uncertainty-aware evaluation for maintenance decision support.

## Overview

This project studies Remaining Useful Life prediction on NASA C-MAPSS turbofan engine degradation data using Bi-LSTM and CNN-BiLSTM models.

The project focuses on:
- multivariate time-series modeling
- asymmetric loss design using LinEx loss
- predictive uncertainty estimation with Monte Carlo Dropout
- decision-relevant evaluation for maintenance risk analysis

## Methods

- CNN-BiLSTM
- MSE Loss
- LinEx Loss
- Monte Carlo Dropout

## Evaluation Metrics

- RMSE
- MAE
- NASA Score
- Overestimation Rate

## Current Status

This repository is being organized into a clean public version for project presentation and reproducible experiments.

## Research Focus

The goal is to connect prediction performance, uncertainty estimation, and maintenance decision-making under asymmetric operational risk.
